Núcleo QRGB

Descargar módulo

Este es el módulo central que implementa la generación de códigos QR multicapa. Puedes integrarlo en cualquier proyecto Python.

"""
QRGB - Generador de Códigos QR Multicapa
Desarrollado por Federico Anderson con asistencia de Claude Sonnet 3.5
"""

import qrcode
import numpy as np
from PIL import Image

class QRGBGenerator:
    """Generador de códigos QR con múltiples capas de información en RGB"""
    
    def __init__(self):
        """Inicializa el generador de códigos QRGB"""
        self.qr_version = 1  # Versión predeterminada de QR
        self.error_correction = qrcode.constants.ERROR_CORRECT_M  # Nivel de corrección de errores
    
    def set_version(self, version):
        """Establece la versión del código QR (1-40)"""
        if 1 <= version <= 40:
            self.qr_version = version
        else:
            raise ValueError("La versión debe estar entre 1 y 40")
    
    def set_error_correction(self, level):
        """Establece el nivel de corrección de errores"""
        levels = {
            'L': qrcode.constants.ERROR_CORRECT_L,  # ~7% de corrección
            'M': qrcode.constants.ERROR_CORRECT_M,  # ~15% de corrección
            'Q': qrcode.constants.ERROR_CORRECT_Q,  # ~25% de corrección
            'H': qrcode.constants.ERROR_CORRECT_H   # ~30% de corrección
        }
        if level in levels:
            self.error_correction = levels[level]
        else:
            raise ValueError("Nivel de corrección no válido. Use 'L', 'M', 'Q' o 'H'")
    
    def generate_qr_layer(self, data, color=(0, 0, 0)):
        """Genera una capa de código QR para un canal de color específico"""
        qr = qrcode.QRCode(
            version=self.qr_version,
            error_correction=self.error_correction,
            box_size=10,
            border=4,
        )
        qr.add_data(data)
        qr.make(fit=True)
        
        # Crear imagen QR en el canal especificado
        img = qr.make_image(fill_color=color, back_color="white")
        return img
    
    def generate_rgb_qr(self, red_data, green_data, blue_data):
        """Genera un código QR multicapa combinando tres mensajes en los canales RGB"""
        # Generar códigos QR para cada canal
        red_qr = self.generate_qr_layer(red_data, color=(255, 0, 0))
        green_qr = self.generate_qr_layer(green_data, color=(0, 255, 0))
        blue_qr = self.generate_qr_layer(blue_data, color=(0, 0, 255))
        
        # Convertir a arrays de numpy
        red_array = np.array(red_qr)
        green_array = np.array(green_qr)
        blue_array = np.array(blue_qr)
        
        # Asegurar que todas las imágenes tienen el mismo tamaño
        min_height = min(red_array.shape[0], green_array.shape[0], blue_array.shape[0])
        min_width = min(red_array.shape[1], green_array.shape[1], blue_array.shape[1])
        
        # Recortar al tamaño mínimo
        red_array = red_array[:min_height, :min_width]
        green_array = green_array[:min_height, :min_width]
        blue_array = blue_array[:min_height, :min_width]
        
        # Combinar los canales
        rgb_array = np.zeros((min_height, min_width, 3), dtype=np.uint8)
        
        # Establecer los canales R, G y B basados en los códigos QR
        rgb_array[..., 0] = red_array[..., 0]  # Canal rojo
        rgb_array[..., 1] = green_array[..., 1]  # Canal verde
        rgb_array[..., 2] = blue_array[..., 2]  # Canal azul
        
        # Crear imagen final
        combined_img = Image.fromarray(rgb_array)
        return combined_img
    
    def save_qr(self, image, filename):
        """Guarda la imagen QR en un archivo"""
        image.save(filename)
        return filename

# Ejemplo de uso
if __name__ == "__main__":
    generator = QRGBGenerator()
    generator.set_error_correction('H')  # Máxima corrección de errores
    
    # Datos para cada capa
    red_data = "Datos para el canal rojo: https://ejemplo.com/rojo"
    green_data = "Información en canal verde: contacto@ejemplo.com"
    blue_data = "Datos en azul: +1234567890"
    
    # Generar código QR multicapa
    qr_image = generator.generate_rgb_qr(red_data, green_data, blue_data)
    
    # Guardar imagen
    generator.save_qr(qr_image, "qrgb_ejemplo.png")
    print("Código QR multicapa generado con éxito!")

Características principales:

  • Capas independientes: Cada canal de color (RGB) contiene información diferente
  • Configurable: Ajusta la versión del QR y nivel de corrección de errores
  • Compatible: Los códigos generados son escaneables por aplicaciones estándar de QR
  • Extensible: Fácil de integrar en aplicaciones web o de escritorio
Nota: Para escanear una capa específica, utiliza un filtro de color correspondiente (rojo, verde o azul).

Implementación Streamlit

La versión Streamlit ofrece una interfaz web interactiva para generar códigos QRGB con visualización en tiempo real.

"""
QRGB - Aplicación Streamlit para generación de códigos QR multicapa
Desarrollado por Federico Anderson con asistencia de Claude Sonnet 3.5
"""

import streamlit as st
import numpy as np
from PIL import Image
import qrcode
import io
from qrgb import QRGBGenerator  # Importamos el módulo principal

def main():
    # Configuración de la página
    st.set_page_config(
        page_title="QRGB - Generador de Códigos QR Multicapa",
        page_icon="🔍",
        layout="wide"
    )
    
    # Título y descripción
    st.title("🔍 QRGB - Generador de Códigos QR Multicapa")
    st.markdown("""
    Esta aplicación te permite generar códigos QR que contienen **tres mensajes diferentes** 
    en las capas de color rojo, verde y azul. Cada mensaje puede ser escaneado 
    individualmente usando un filtro del color correspondiente.
    """)
    
    # Crear instancia del generador
    generator = QRGBGenerator()
    
    # Formulario para la configuración
    with st.form("qrgb_form"):
        st.subheader("📝 Datos para cada capa")
        
        # Entrada de datos para cada canal
        col1, col2, col3 = st.columns(3)
        
        with col1:
            st.markdown("### 🔴 Canal Rojo")
            red_data = st.text_area("Datos para capa roja", 
                                   "https://ejemplo.com/rojo", 
                                   height=100,
                                   help="Este texto se codificará en el canal rojo")
        
        with col2:
            st.markdown("### 🟢 Canal Verde")
            green_data = st.text_area("Datos para capa verde", 
                                     "contacto@ejemplo.com", 
                                     height=100,
                                     help="Este texto se codificará en el canal verde")
        
        with col3:
            st.markdown("### 🔵 Canal Azul")
            blue_data = st.text_area("Datos para capa azul", 
                                    "+1234567890", 
                                    height=100,
                                    help="Este texto se codificará en el canal azul")
        
        # Opciones avanzadas
        st.subheader("⚙️ Configuración")
        col1, col2 = st.columns(2)
        
        with col1:
            qr_version = st.slider("Versión del código QR", 
                                  min_value=1, max_value=40, value=5,
                                  help="Versiones más altas permiten más datos pero crean códigos más grandes")
        
        with col2:
            error_correction = st.selectbox("Nivel de corrección de errores",
                                           options=["L", "M", "Q", "H"],
                                           index=2,
                                           help="H ofrece la mayor corrección de errores (30%)")
        
        # Botón para generar
        submit_button = st.form_submit_button("✨ Generar Código QR")
    
    # Procesar cuando se hace clic en el botón
    if submit_button:
        try:
            with st.spinner("Generando código QR multicapa..."):
                # Configurar el generador
                generator.set_version(qr_version)
                generator.set_error_correction(error_correction)
                
                # Generar el código QR combinado y cada capa individual
                combined_qr = generator.generate_rgb_qr(red_data, green_data, blue_data)
                red_qr = generator.generate_qr_layer(red_data, color=(255, 0, 0))
                green_qr = generator.generate_qr_layer(green_data, color=(0, 255, 0))
                blue_qr = generator.generate_qr_layer(blue_data, color=(0, 0, 255))
                
                # Mostrar resultados
                st.subheader("🎯 Resultado")
                
                col1, col2, col3, col4 = st.columns(4)
                
                with col1:
                    st.markdown("### 🔴🟢🔵 Combinado")
                    st.image(combined_qr, use_column_width=True)
                    
                    # Opción para descargar
                    buf = io.BytesIO()
                    combined_qr.save(buf, format='PNG')
                    st.download_button(
                        label="⬇️ Descargar QR Combinado",
                        data=buf.getvalue(),
                        file_name="qrgb_combined.png",
                        mime="image/png"
                    )
                
                with col2:
                    st.markdown("### 🔴 Capa Roja")
                    st.image(red_qr, use_column_width=True)
                    
                    buf = io.BytesIO()
                    red_qr.save(buf, format='PNG')
                    st.download_button(
                        label="⬇️ Descargar Capa Roja",
                        data=buf.getvalue(),
                        file_name="qrgb_red.png",
                        mime="image/png"
                    )
                
                with col3:
                    st.markdown("### 🟢 Capa Verde")
                    st.image(green_qr, use_column_width=True)
                    
                    buf = io.BytesIO()
                    green_qr.save(buf, format='PNG')
                    st.download_button(
                        label="⬇️ Descargar Capa Verde",
                        data=buf.getvalue(),
                        file_name="qrgb_green.png",
                        mime="image/png"
                    )
                
                with col4:
                    st.markdown("### 🔵 Capa Azul")
                    st.image(blue_qr, use_column_width=True)
                    
                    buf = io.BytesIO()
                    blue_qr.save(buf, format='PNG')
                    st.download_button(
                        label="⬇️ Descargar Capa Azul",
                        data=buf.getvalue(),
                        file_name="qrgb_blue.png",
                        mime="image/png"
                    )
                
                # Mostrar instrucciones de uso
                st.success("¡Código QR multicapa generado con éxito!")
                
                with st.expander("📋 Instrucciones para escanear"):
                    st.markdown("""
                    ### Cómo escanear las diferentes capas:
                    
                    1. **Capa Roja**: Usa un filtro rojo sobre la cámara de tu dispositivo
                    2. **Capa Verde**: Usa un filtro verde sobre la cámara
                    3. **Capa Azul**: Usa un filtro azul sobre la cámara
                    4. **Código Combinado**: Puede ser difícil de escanear; usa las capas individuales para mayor fiabilidad
                    
                    *Consejo: Puedes crear filtros simples con papel celofán de colores o usar aplicaciones de filtro de color en tu dispositivo.*
                    """)
        
        except Exception as e:
            st.error(f"Error al generar el código QR: {e}")
    
    # Información adicional
    st.markdown("---")
    st.markdown("### 📚 Acerca de QRGB")
    st.markdown("""
    QRGB es un proyecto de código abierto que extiende la funcionalidad de los códigos QR estándar 
    para permitir almacenar tres mensajes independientes en un solo código.
    
    Desarrollado por Federico Anderson con asistencia de Claude Sonnet 3.5 de Anthropic.
    """)

if __name__ == "__main__":
    main()

Implementación Flask

La versión Flask proporciona una aplicación web completa con backend en Python, ideal para despliegues en servidores con más funcionalidades:

Estructura de archivos
qrgb_flask/
├── app.py              # Aplicación principal Flask
├── qrgb.py             # Módulo principal QRGB
├── routes.py           # Definición de rutas
├── static/
│   ├── css/
│   │   └── styles.css
│   ├── js/
│   │   └── qrgb.js
│   └── images/
├── templates/
│   ├── base.html
│   ├── index.html
│   └── result.html
└── requirements.txt    # Dependencias
Características
  • API REST: Endpoints para generar QR programáticamente
  • Persistencia: Opción de guardar códigos generados
  • Interfaz web: Frontend amigable para usuarios
  • Estadísticas: Seguimiento de códigos generados
  • Seguridad: Validación de entradas

Instrucciones de instalación:

# Clonar o descargar el repositorio
# Navegar al directorio de la aplicación

# Crear un entorno virtual (recomendado)
python -m venv venv
source venv/bin/activate  # En Windows: venv\Scripts\activate

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar la aplicación
python app.py

# La aplicación estará disponible en http://localhost:5000

Opciones de despliegue

La aplicación Flask puede desplegarse en múltiples plataformas:

  • Replit: Importa el código y ejecuta desde la plataforma
  • PythonAnywhere: Ideal para hosting gratuito con bajo tráfico
  • Heroku: Escalable para aplicaciones con más tráfico
  • AWS/GCP/Azure: Para despliegues de nivel empresarial
